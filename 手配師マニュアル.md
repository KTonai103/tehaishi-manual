## 予定調整

### 手術日程・入院決定フロー

手術日程決定 → 入院日決定

### 入院ルール

- 1日4人まで（個室・男子部屋・女子部屋に注意）
- 透析患者は同時に5人まで（緊急入院考慮し、なるべく4枠以下で運用）
- 基本10時入院。遠方の場合は午後入院も可。持参薬履歴がある人は休日入院も可
- 西9師長と相談しながらベッドコントロール

### 手術症例のスケジュール確認タイミング

- 翌週分 → 週前半に再確認
- 翌々週分 → 週後半までにおおよそ確定

**早めに入院させる症例（要注意）**：ヘパリン置換、血糖コントロール、TEE含め術前検査不足症例 等

## 患者連絡・入院予定登録

### 患者連絡のタイミング

- 基本1週間前（カンファのタイミング等にもよる。早めに連絡すべき症例は事前把握）

### 患者連絡時の確認事項

以下の項目を患者に確認し、カルテ記録を行う（患者手配業務提携簿の定型文を使用）：

- **アレルギー歴**（薬物・食物・喘息）
- **抗血小板薬・抗凝固薬の内服確認と休薬指示**
- **糖尿病治療薬の確認**
- **術前歯科外来受診について**：抜歯の可能性があることも説明
- **入院時間**：9〜10時に1階入退院センターへ来てもらうよう伝える
- **部屋希望**：入院時説明で案内されているが、再度個室か大部屋かを確認する

## 他科外来予約

### 麻酔科

- **入院中外来を優先**する（術前外来より入院中からが優先）
- 麻酔科外来の前日12:30締め切り。3ヶ月以内の採血・Xp・ECGが必要

### 他科コンサルト共通

- 紹介部分の横の定型文ボタンから定型文を入力する
- EVARの場合は**EVAR用の定型文**を使用する

### 透析患者

- **入院日決定前**に、コミュニケートで**腎臓内科 酒井先生にメール**し、透析日程を調整
- 心臓外科の透析枠は**5枠**しかないため、酒井先生と相談（なるべく4枠以下で運用）
- 入院決定後、**かかりつけ透析病院に連絡**し、透析サマリ等を透析室にFAX依頼（06-6879-6049）
- 基本的に入院日の透析は不可（緊急を除く）のため、入院日・オペ日・透析日を調整

### 術前の抗血小板・抗凝固薬

- 適宜Aクラスに確認。休薬期間は手配師横の資料参照

| 症例 | 方針 |
|------|------|
| 開心術 | 基本全て休薬 |
| CAD/PCI後 | SAPTのみ継続 |
| カテーテル治療（deb TEVAR含む） | 休薬不要のことも多い |
| RIBS | 全休薬 |

**注意事項：**
- MICS前の傍脊椎ブロック（パラバー）やGR前の硬膜外麻酔（epi）は、より早めの休薬が必要
- 胸腹部手術（オープンもPMEGも）はスパイナルドレナージを常に想定し、術前+1の休薬必要
- イコサペント酸エチル（7〜10日）、クロピドグレル（14日）等は特に長いので要注意

### 術前の糖尿病治療薬

- 休薬期間は手配師横の資料参照
- 術前（できれば1ヶ月前まで）に、紹介基準（手配師横の貼紙）に則って、糖内外来受診させる
- 未受診の場合：日未定で紹介状作成。本人に連絡し、午前11時までに各科外来へ直接受診

## 手術申し込み

**〆切：3営業日前の午前11時**（基本4営業日前に済ませる）  
〆切以降の変更・差替は麻酔科ライターとOP師長さんの電話許可が必要  
予定の締め切り前の手術申し込みは**手術依頼入力**から行う

### 枠の注意事項

- MICS、DaVinci、redo手術は原則1日枠。MIDCAB＊は半日枠。小児のredoは0.7日枠扱い
- 人工心肺は2台まで。3台目は超緊急用
- 火曜・木曜は開心術 + OR3かOR2/7（人工心肺は1台まで）
- 手術部運営ルール：2例目以降の入室時間が下記を超えると当日急遽キャンセル
  - 14時：開心術
  - 16時：endo/TAVI
  - 18時：局麻の小手術

### 申込項目

#### 日時・時間
- 8:30〜16:45の間で帯を選択。手術の間は30分あける。小児は朝1例目
- TAVI：8:45〜11:00 / 11:30〜14:00 / 14:30〜17:00

#### 術式
- Redoの場合は術式・コメントで明記
- debranch TEVARは1/2/totalを明記

#### 体位

| 術式 | 体位 |
|------|------|
| 基本 | 仰臥位 |
| MICS-A/M | 左半側臥位 |
| MICS CABG/MICS-LAAc/下行置換/TAAA | 右半側臥位 |
| PDA ligation | 右側臥位（左が上） |
| PA banding | 仰臥位 |

#### 麻酔法
- 麻酔科依頼：全麻/ブロック→あり、静麻/局麻→なし
- 側開胸症例（TA-AVI, MICS, シート移植）：全麻 + 神経ブロック
- 開腹手術：全麻 + Epi
- 骨格筋採取：静脈麻酔 + 大腿神経ブロック
- TAVI/EVAR/TEVAR：「全麻または局麻」（TAVIアプローチ部位や麻酔法は付箋参照）

#### 希望手術室
- TAVI/TEVAR/EVAR：OR1
- PMEG：OR2（火曜は必ずハイブリッド手術室枠あり）
- Harmony（TPVR）の場合：「出張」+フリーコメントに「OP室キープ」

#### 術者
- 術者：ライター
- 内科TAVI → 受け持ち：松廣Dr. 術者：中村Dr.

#### 入力責任者 / 問い合わせ連絡先
- 「手配師名・7600」
